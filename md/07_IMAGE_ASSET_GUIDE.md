# 07 — Image Asset Guide

## Goal

The personal images must make the portfolio feel like a campaign, not a selfie gallery.

Do not upload random photos.

Every image should support a section, a transition, or the brand mood.

## Current main asset

The main hero asset is a cinematic 16:9 night portrait of Gurtej seated on an orange bench with greenery, red turban, white shirt, and strong warm light.

This is the identity anchor.

## Required image count

Minimum: **6 strong images**

Ideal: **8–10 strong images**

Do not use 20 average images.

## Required shots

### 1. Hero portrait

Status: already available.

Use:

- homepage hero
- loading transition
- social preview crop
- about background crop

Requirements:

- 16:9
- 4K or high-resolution
- cinematic color grade
- face clear
- enough negative space for text

### 2. Close-up portrait

Purpose:

- about section
- philosophy section
- profile hover/crop

Direction:

- face closer
- strong eye focus
- dark or clean background
- same red turban if possible
- DSLR-style depth

### 3. Standing/full-body editorial shot

Purpose:

- section transition
- about visual
- scroll story

Direction:

- full body
- strong posture
- cinematic framing
- not casual selfie
- clean background

### 4. Working shot

Purpose:

- credibility
- process section

Direction:

- laptop/design/coding setup
- low light or warm desk light
- hands visible
- screen glow
- not messy hostel/room background unless styled

### 5. Silhouette/back shot

Purpose:

- visual transition
- mood section
- contact background

Direction:

- back facing camera
- strong outline
- dark background
- cinematic light source

### 6. Motion shot

Purpose:

- scroll/parallax sequence
- visual experiment section

Direction:

- walking or turning
- slight motion blur
- confident but natural
- controlled lighting

### 7. Detail shot

Purpose:

- texture, luxury feel

Options:

- hand on keyboard
- watch/kara/turban fabric
- laptop close-up
- hand holding phone
- shoes/bench/environment detail

### 8. Optional environmental shot

Purpose:

- background texture

Direction:

- same plants/bench/location
- no face required
- cinematic empty frame
- useful for overlays

## Image quality requirements

Every image must be:

- sharp
- high resolution
- not over-smoothed
- not heavily AI-looking
- not overexposed
- not too saturated
- consistent color grade

## Color grading rules

Apply:

- deep blacks
- warm highlights
- controlled skin tone
- slightly muted greens
- strong but not oversaturated red
- restrained orange
- subtle film grain

Avoid:

- Instagram filters
- blue neon overkill
- HDR face
- fake skin smoothing
- random color tones across photos

```text
gurtej-environment-night.webp
```

## Crops needed

For each major image, create:

- desktop wide: 16:9
- mobile vertical: 4:5 or 9:16
- square thumbnail: 1:1 if needed
- social preview: 1200x630

## Hero mobile problem

The 16:9 image will not automatically work on mobile.

Create a mobile crop where:

- face is visible
- body posture still readable
- text can sit at bottom
- orange bench still appears partially
- no important detail is cut badly

## Project screenshots

For each project, capture:

- desktop homepage screenshot
- mobile screenshot
- one detail/section screenshot
- optional animated/video preview later

Naming:

```text
project-sws-luxury-desktop.webp
project-sws-luxury-mobile.webp
project-ghost-engineer-desktop.webp
project-ghost-engineer-mobile.webp
project-obsidian-desktop.webp
project-obsidian-mobile.webp
project-iron-forge-desktop.webp
project-iron-forge-mobile.webp
```

## Where images should be used

| Image | Section |
|---|---|
| Hero portrait | Hero |
| Close-up | About / Philosophy |
| Full body | Transition / About |
| Working shot | Process |
| Silhouette | Contact / Manifesto |
| Motion shot | Visual experiments |
| Detail shot | Texture / Loading / Divider |
| Environment | Background overlays |

## Hard rule

All images must look like they belong to the same shoot.

If one image looks bright and casual while another looks cinematic, the portfolio will lose its identity.
