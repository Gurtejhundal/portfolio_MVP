# 00 — Master Brief

## Project

Personal portfolio website for **Gurtej**.

## Objective

Build an award-level portfolio website that positions Gurtej as a **creative developer / digital experience engineer** rather than a generic student developer.

The website must use a cinematic full-image hero, strong typography, premium dark visual language, and controlled scroll-based animation.

## Core positioning

Primary label:

> Creative Developer

Secondary labels:

> Motion-led interfaces  
> Premium landing pages  
> Digital product experiences

Avoid making the website feel like a normal resume website.

Do not lead with:

- B.Tech student
- frontend developer
- full stack developer
- HTML/CSS/JS skills
- skill bars
- generic about-me paragraphs

Lead with taste, execution, and project quality.

## Strategic truth

The user has built several visually driven production projects, including:

- Ghost Engineer — open-source impact lab / blueprint generator
- SWS Luxury — luxury commerce experience for attars, perfumes, jewelry, gifting
- Obsidian Finish Studio — premium car detailing website
- Iron Forge Athletics — premium gym website

This pattern is the actual brand signal: premium, commercial, cinematic, conversion-aware digital experiences.

## Main creative concept

**“Presence Engine”**

The portfolio should feel like the visitor is entering a cinematic personal system: image, typography, movement, and project proof working together.

The hero image is not decoration. It is the identity anchor.

The site should feel like:

- luxury editorial
- cinematic product launch
- digital experience studio
- personal brand campaign

It should not feel like:

- college resume
- template portfolio
- random animation showcase
- GitHub project list
- overly playful developer site

## Tone

Use short, sharp, confident copy.

Do not write long explanations. Use strong fragments where appropriate.

Example tone:

> I build digital experiences that feel deliberate before they feel technical.

> Motion is not decoration. It is hierarchy in time.

> Luxury is restraint. Interaction is rhythm. Interfaces should carry presence.

## Experience goals

The visitor should feel:

1. This person has taste.
2. This person understands premium presentation.
3. This person can build production websites.
4. This person is not copying common developer templates.
5. This person can handle visual systems, motion, and commercial structure.

## Site format

Single-page portfolio with optional project detail overlays/pages.

Primary sections:

1. Loader
2. Hero
3. Identity / Manifesto
4. Selected Work
5. Process / Principles
6. Visual Experiments
7. About Snapshot
8. Contact

## Non-negotiables

- Full-viewport cinematic hero
- Large typography
- Dark premium base
- Scroll-driven motion
- Mobile-first polish
- No skill bars
- No childish icon grid
- No generic gradient cards
- No fake overclaiming
- No excessive text
- No overuse of 3D
- No animation without purpose

## Award-level criteria

The site must succeed on:

- first-screen impact
- typography
- motion restraint
- image treatment
- mobile experience
- project storytelling
- section pacing
- visual consistency
- performance

The site should have one or two unforgettable motion moments, not constant noise.
