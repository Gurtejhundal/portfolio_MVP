# 14 — Reference Direction

## Purpose

This document defines what kind of inspiration to study and what to avoid.

Do not copy references directly. Extract principles.

## Portfolio direction to study

Study websites that use:

- fullscreen imagery
- strong typography
- cinematic pacing
- restrained motion
- premium negative space
- high-quality transitions
- project storytelling
- editorial layout

## Good inspiration categories

### 1. Cinematic studio sites

Study for:

- transitions
- dark atmosphere
- pacing
- interaction confidence

Examples to research:

- Active Theory
- Lusion
- Locomotive
- Resn
- Immersive Garden

### 2. Premium personal portfolios

Study for:

- typography hierarchy
- personal positioning
- project presentation
- controlled motion

Examples to research:

- Bruno Simon
- Daniel Spatzek
- Patrick David
- awwwards portfolio winners

### 3. Luxury brand/editorial sites

Study for:

- image treatment
- restraint
- spacing
- copy brevity
- product presence

Examples:

- fragrance/luxury e-commerce experiences
- fashion editorials
- premium automotive sites

## What to extract

Extract:

- section pacing
- how text enters
- how images are revealed
- use of negative space
- how projects are introduced
- how contact sections close the site
- typography scale
- mobile simplification

Do not copy:

- exact layouts
- exact animations
- logos
- project names
- visual assets
- text content

## Hero references to look for

Search terms:

```text
awwwards fullscreen portrait portfolio
cinematic personal portfolio hero
editorial developer portfolio
motion portfolio hero image typography
```

Look for:

- portrait integrated with text
- large name typography
- dark overlays
- image as full environment
- slow reveal
- minimal copy

## Main danger

The user wants “heavenly animated each and everything.”

That instinct is dangerous.

The website should feel animated because the few important things move with precision, not because every element is moving.

## Award-level principle

Award-level design is not maximum complexity.

It is maximum control.

Use this rule:

> If an animation does not improve hierarchy, emotion, or understanding, remove it.

## Reference evaluation checklist

When studying a reference, ask:

1. What is the first 3-second impression?
2. Where does the eye go first?
3. How much text is visible?
4. What is the main motion idea?
5. Is the animation helping or distracting?
6. Does mobile still feel premium?
7. Can this idea be adapted to Gurtej's image and projects?

## Direction summary

The final portfolio should feel closer to:

```text
cinematic campaign + premium project archive + developer identity system
```

Not:

```text
resume website + animated cards + tech stack grid
```
